# Features
- World generation
    - Trees
    - Mushrooms
- The map actually looks like a giant island
- Player movement
    - WASD
- Player physics
    - Collision
    - Different speed with different blocks
- UI
  - Buttons
  - Input fields
  - Forms
- World saving/loading
- Placing blocks
    - Hold F to place blocks under you
- Inventory
    - Press C to pick block under the player

### Plans for the future
- Multiplayer
